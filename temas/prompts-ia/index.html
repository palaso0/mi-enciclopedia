Prompts para IA

En resumen: Un prompt es el texto que le das a una IA para que haga algo. No es magia, son instrucciones. Cuanto mejor las escribas, mejor será el resultado.

1. Qué es un prompt

Un prompt es simplemente el texto que le escribes a una IA. Cuando abres ChatGPT y escribes algo, eso es un prompt. No hay nada misterioso: es una instrucción en lenguaje natural.

La diferencia entre un prompt bueno y uno malo es la diferencia entre decirle a alguien "haz comida" y decirle "prepara una pasta al pesto con pollo a la plancha, sin lácteos". La IA puede hacer cualquier cosa, pero necesita que le digas qué.

2. Anatomía de un buen prompt

Contexto

Dale a la IA información de fondo. Quién eres, qué necesitas, para qué es.

Tarea

Dile explícitamente qué quieres que haga. Resumir, crear, analizar, traducir, etc.

Formato

¿Quieres una lista, un párrafo, código, una tabla? Pídelo.

Restricciones

Límites: tono formal, máximo 200 palabras, sin jerga técnica, etc.


// Prompt genérico (malo)
"Resumime esto"

// Prompt completo (bueno)
"Soy estudiante de derecho. Necesito un resumen de este artículo
sobre contratos digitales. Que sea de 3 párrafos máximo, usando
lenguaje claro sin tecnicismos. Al final, pon los 3 puntos
más importantes en una lista."

3. Tipos de prompts

Zero-shot

sin ejemplos

Le das la tarea directamente sin mostrar ejemplos. Es el tipo más básico.


"Clasifica esta reseña como positiva, negativa o neutra:
'El producto llegó roto y nadie me respondió el reclamo'"

Few-shot

con ejemplos

Le muestras uno o más ejemplos de cómo quieres la respuesta. La IA aprende el patrón y lo replica.


"Clasifica reseñas como positivas, negativas o neutras.

Ejemplo 1: '¡Amo este producto!' → Positiva
Ejemplo 2: 'No funciona para nada' → Negativa
Ejemplo 3: 'Es okay, nada especial' → Neutra

Ahora clasifica: 'El producto llegó roto y nadie me respondió'"

Chain-of-Thought (CoT)

razonamiento paso a paso

Pides que la IA razone antes de dar la respuesta final. Mejora mucho los resultados en problemas de lógica o matemática.


"Piensa paso a paso antes de responder:

Si tengo 3 camisas y 4 pantalones, ¿cuántos outfits
distintos puedo armar?"

Role Prompting

asignar un rol

Le asignas un rol o personalidad a la IA. Esto cambia el tono, vocabulario y profundidad de las respuestas.


"Eres un profesor de física. Explica la relatividad
a un niño de 10 años. Usa analogías cotidianas,
nada de fórmulas."

ReAct (Reasoning + Acting)

razonar y actuar

La IA alterna entre razonar y ejecutar acciones (como buscar información o usar herramientas). Es la base de los agentes de IA.


"Piensa paso a paso. Si necesitas información que no tienes,
di qué buscarías y por qué. Después continúa con el razonamiento."

4. Técnicas avanzadas

Prompt de system

instrucción permanente

En APIs como OpenAI, el "system prompt" es una instrucción que siempre está presente. Define el comportamiento base de la IA antes de que el usuario escriba algo.


// System prompt
"Eres un asistente técnico. Responde siempre en español,
usa ejemplos de código, y si no sabes algo, dilo."

// User prompt
"¿Cómo hago un fetch en JavaScript?"

Prompt chaining

varios prompts encadenados

Divides una tarea compleja en pasos. Cada paso usa el resultado del anterior como input.

1 "Resume este artículo de 2000 palabras en 5 puntos clave"
2 "Toma estos 5 puntos y explica cada uno con un ejemplo cotidiano"
3 "Convierte eso en una presentación de diapositivas con títulos y subtítulos"

Delimitadores

separar instrucciones de datos

Usas caracteres especiales para decirle a la IA dónde termina tu instrucción y empiezan los datos. Evita que la IA confunda contenido con instrucciones.


"Traduce el siguiente texto al inglés. Usa triple comilla
como delimitador:

'''La inteligencia artificial está transformando todas
las industrias, desde la medicina hasta la agricultura.'''"

5. Errores comunes

MAL: Ambiguo

"Explicame esto"

¿Qué es "esto"? ¿Para quién? ¿A qué nivel de profundidad?

BIEN: Específico

"Explicame cómo funciona un transformador en lenguaje simple, para alguien que sabe programar pero no sabe de machine learning"

MAL: Sobrecargado

"Hazme un resumen, después tradúcelo, después hazlo más corto, después ponlo en formato de lista, y agrégame un ejemplo"

Demasiado en un solo prompt. La calidad baja.

BIEN: Encadenado

Paso 1: "Resume esto en 3 párrafos"
Paso 2: "Traduce el resumen al inglés"
Paso 3: "Convierte esto en una lista de viñetas"

MAL: Sin contexto

"Escribe un email profesional"

¿Para quién? ¿Sobre qué? ¿Qué tono?

BIEN: Con todo el contexto

"Escribe un email formal para un cliente que se quejó por un retraso en la entrega. Explica que hubo un problema logístico, que ya se resolvió, y que el pedido llega mañana. Tono profesional pero empático."

6. Template de prompt

Si no sabes por dónde empezar, usa esta estructura como base:


[SOS / CONTEXTO]
Eres un profesor de matemáticas con 20 años de experiencia.

[TAREA]
Explícame qué es una derivada.

[FORMATO]
Usa una analogía cotidiana primero, después la definición
técnica, y al final un ejemplo resuelto.

[RESTRICCIONES]
Que sea comprensible para un estudiante de 16 años
que nunca vio cálculo. Máximo 300 palabras.

La regla de oro: Si la IA te da una respuesta mala, el 90% de las veces el problema es el prompt, no la IA. Antes de cambiar de herramienta, mejora la instrucción.
